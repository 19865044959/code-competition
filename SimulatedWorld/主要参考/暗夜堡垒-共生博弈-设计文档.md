## 一、游戏总览

### 1.1 核心概念

两名玩家在 25×25 方格地图上各控制 3 名英雄（采集/建造英雄×2 + 开拓者×1），白天采集建造、攻略副本、执行悬赏任务、外交贸易，夜晚必须回防操控防御塔抵御递增强度的兽潮。10 天（2000 回合）后按累计积分定胜负。

地图资源分布不均——**铜矿只有A能采、硅矿只有B能采**（阵营资源绑定），贸易站是获取对方独占资源的唯一途径。芯片=铜+硅，是满级建筑的必须材料。不合作=造不出芯片=永远卡在Lv3=后期兽潮无法抗衡。

### 1.2 对局基础配置

| 参数 | 数值 |
|------|------|
| 对局模式 | 1V1，双方出生点地图对角对称分布 |
| 对局周期 | 10 游戏天 |
| 回合概念 | **1 天 = 200 回合**（白天 120 回合，黑夜 80 回合），10 天 = 2000 回合 |
| 回合定义 | 每回合 = 判题器发送当前地图信息 → 玩家收到后返回本回合动作 → 判题器执行并反馈结果 |
| 每队英雄数 | 3 名（采集/建造英雄×2、开拓者×1） |
| 英雄阵亡 | 次日在基地复活 |
| 淘汰条件 | 基地被摧毁 → 直接淘汰，后续无法获取积分 |
| 地图规格 | 25×25 正方形网格 |
| 双方关系 | **竞争&合作**——不可直接攻击对方，通过贸易互换材料+积分排名竞争 |

### 1.3 核心循环

```
【白天阶段】（120 回合）
├── 采集资源、建造升级、维修围墙
├── 2名采集英雄组队挑战 PVE 副本
├── 开拓者外出执行悬赏任务（探索、使用物品）
├── 开拓者去地图中心贸易站发起交换
└── 黄昏：黑夜临近预警通知

【黑夜阶段】（80 回合）
├── 兽潮从四面八方来袭，每夜递增
├── 防御塔需要英雄站在旁边（周围1格）操控才能开火（射程2格）
├── 留在野外未回防的英雄 → 被野兽攻击，极大概率阵亡
├── 英雄阵亡 → 次日在基地复活
├── 围墙阻挡兽群前进
├── 天亮 → 阳光彻底杀死地图上所有残余兽群
└── 建筑受损 → 次日需修复
```

### 1.4 10 天节奏

| 阶段 | 天数 | 每夜兽潮 | 核心主题 |
|------|------|---------|---------|
| 新手期 | 第 1 天 | 2-3 只小型兽 | 搭建基础防御、熟悉操作 |
| 发育期 | 第 2-5 天 | 递增至 8-10 只混合兽 | 升级建筑、打副本、执行悬赏任务、初步外交 |
| 中期 | 第 6-8 天 | 10-15 只混合+大型兽 | 获取图纸升级满级建筑、竞争加剧 |
| 末期 | 第 9-10 天 | 15-25 只+BOSS | 极限防守、最终博弈、冲刺积分 |

## 二、地图与地形系统

### 2.1 地图规格

- 全图 **25×25** 标准正方形网格
- 单个地形格子为 1×1 通行单元
- **基地** 占用 **2×2** 网格，其余建筑（研究台、工坊、仓库、农田、围墙、防御塔）占用 **1×1** 网格
- 围墙为 **1×1** 分段
- 两名玩家出生点位于地图对角对称位置

### 2.2 地形分类

| 地形类型 | 网格占比 | 核心资源产出 |
|---------|---------|------------|
| 平坦草地 | 30% | 浆果 |
| 密林区域 | 20% | 木材、草药 |
| 山地矿脉 | 40% | 石料、铜矿、铁矿、硅矿、硫磺 |
| 湖泊/河流 | 10% | 淡水、鱼类 |
| 废弃据点 | 固定点位（1 处） | 一次性挑战副本 |
| 跨空间贸易站 | 地图正中心（1 处） | 唯一的外交贸易交互点 |

### 2.3 阵营资源绑定

- **无领地概念**。地图资源对双方英雄可见、可到达
- **阵营资源绑定**：
  - **铜矿**：只有 **A 阵营**英雄可以采集
  - **硅矿**：只有 **B 阵营**英雄可以采集
  - 其余资源（木材、石料、铁矿、硫磺、食物、草药、淡水）：双方均可采集
- 对方独占资源只能通过**跨空间贸易站**交换获得
- 所有己方建筑和英雄共享视野
- 每个建筑自带基础视野（周围 2 格）

### 2.4 资源分布不对称

| 资源类型 | 玩家 A | 玩家 B | 说明 |
|---------|--------|--------|------|
| 木材 | ★★★★★ | ★★★★★ | **双方均丰富** |
| 石料 | ★★★★☆ | ★★★★☆ | 双方均有，建筑基础材料 |
| 铁矿石 | ★★★★☆ | ★★★★☆ | **双方均可** → 铁锭 → Lv3 围墙/防御塔 |
| 硫磺 | ★★★☆☆ | ★★★☆☆ | **双方均有** → 火药（防御塔 Lv3 需要） |
| 铜矿石 | ✅ **只有A能采** | ❌ | A 独占 → 铜锭 → 晶体管/电路板 → 芯片 |
| 硅矿石 | ❌ | ✅ **只有B能采** | B 独占 → 纯硅 → 电路板 → 芯片 + 激光塔 |
| 食物（浆果/鱼/小麦） | ★★★★★ | ★★☆☆☆ | A 供食物 |
| 草药 | ★★☆☆☆ | ★★★★★ | B 供草药 → 药膏（高级回血品） |

**强制合作锁——芯片链**：
```
芯片 = 晶体管 + 电路板
     = (铜锭 + 铁锭) + (铜锭 + 纯硅)
          ↑A独占  ↑共有     ↑A独占  ↑B独占

→ 铜矿只有A能采、硅矿只有B能采（阵营资源绑定，走到面前也采不了）
→ 铁锭/硫磺双方自给、铜锭只有A能产、纯硅只有B能产
→ 芯片 = 必须通过贸易站交换才能获得的复合产物
→ 没有芯片 = 永远卡在Lv3 = 后期兽潮无法抗衡（第8晚起激光塔/合金城墙刚需）
```

**渐进式合作节奏**：
```
Lv1→Lv2：纯石料/木材（完全自给，不需要合作）
Lv2→Lv3：石料 + 铁锭 + 火药（铁中立、硫磺双方均有→纯自给）
Lv3→Lv4：芯片 + 纯铁 + 纯硅（芯片横跨双方，强制深度合作）
```

## 三、英雄系统

### 3.1 基础规则

- 每名玩家控制 **3 名英雄**：
  - **采集/建造英雄 ×2** — 资源采集、建筑建造、围墙维修、农田耕种、PVE 副本、**夜间操控防御塔**
  - **开拓者 ×1** — 探索隐藏要素、使用特殊物品、执行悬赏任务、**夜间操控防御塔**、**跨空间贸易**
- 英雄阵亡后在己方**基地**次日起复活（夜间阵亡 → 次日清晨复活）
- 每个英雄有独立背包
- 双方英雄**不可互相攻击**（竞争性合作）
- 英雄**无特殊技能**（仅基础属性随等级提升）

### 3.2 采集/建造英雄 ×2

**核心定位**：资源采集、建筑建造、围墙维修、农田耕种、PVE 副本、夜间操控防御塔

| 属性 | 初级 | 熟练 | 精英 |
|------|------|------|------|
| HP | 150 | 150 | 150 |
| 攻击力 | 8 | +30% | +60% |
| 防御力 | 8 | 8 | 8 |
| 移动速度（格/回合） | 1 | 1 | 1 |
| 背包容量 | 20 格 | 25 格 | 35 格 |
| 采集速度 | 基础值 | +25% | +50% |
| 建造效率 | 100% | 100% | 100% |
| 特殊技能 | — | — | — |

**核心技能**：

| 技能 | 消耗 | 效果 |
|------|------|------|
| **采集** | 1 回合 | 在资源地形格上执行采集，持续至产出完成。完成后物品自动放入背包。详见 3.8.1 |
| **建造** | 持续 N 回合 | 在空地格上建造/升级建筑，中途不可离开。材料在完成时从背包扣除。详见 3.8.2 |
| **维修** | 1 回合 | **仅白天可用**。维修受损建筑，按血量比例消耗材料，维修后建筑恢复满血。详见 3.6 |

**核心职责**：

- 采集所有类型资源
- 建造所有建筑、升级建筑
- 维修所有受损建筑（按血量比例消耗材料）
- 开垦和维护农田（种植小麦或大米）
- 组队挑战 PVE 副本（2 名采集英雄组队）
- **夜晚站在防御塔旁（1 格内）操控防御塔攻击兽群**

### 3.3 开拓者

**核心定位**：执行悬赏任务（探索+使用）、跨空间贸易

| 属性 | 初级 | 熟练 | 精英 |
|------|------|------|------|
| HP | 130 | 130 | 130 |
| 攻击力 | 6 | 6 | 6     |
| 防御力 | 6 | 6 | 6 |
| 移动速度（格/回合） | 5 | +30% | +60% |
| 背包容量 | 10 格 | 15 格 | 20 格 |
| 特殊技能 | — | — | — |

**核心技能**：

| 技能 | 消耗 | 效果 |
|------|------|------|
| **探索** | 2 回合驻留 | 深度探明当前坐标周围区域隐藏内容：配方碎片、宝箱、传送符文、悬赏任务线索等。判题器以自然语言返回探索结果。可能空手而归（概率受等级影响） |
| **使用** | 1 回合 | 在指定坐标使用背包中的物品（如"将特效药倒入湖泊"、"在石碑前放置祭品"、"在卷轴凹槽中放入芯片"） |
| **贸易** | 1 回合 | 在跨空间贸易站发起/响应交易请求。交易物资通过仓库隔空传送，不经过背包 |

**核心职责**：
- 唯一可执行悬赏任务中"探索"和"使用"动作的英雄
- 唯一可发起跨空间贸易的英雄
- 不参与采集、建造（专注任务+贸易）
- 夜间也可操控防御塔（非最优选择）

### 3.4 贸易规则

- 开拓者走到地图中心**跨空间贸易站** → 发起交易请求
- 对方开拓者**不需要在场**（远程响应）
- 交易界面显示对方当前各项资源库存量
- 双方确认交换清单 → 各自仓库物品**隔空瞬间交换**
- 贸易站不可被摧毁、不可被占据

### 3.5 背包与仓库容量

| 容器 | 初始容量 | 升级方式 |
|------|---------|---------|
| 采集英雄背包 | 20 格 | 随等级提升至 35 格 |
| 开拓者背包 | 10 格 | 随等级提升至 20 格 |
| 营地仓库 | 80 格 | 升级仓库建筑：Lv2 → 120 格，Lv3 → 200 格 |

> 仓库存取规则详见 [5.5 仓库](#55-仓库11)。

### 3.6 建筑维修规则

采集英雄可在白天维修受损的建筑（围墙、防御塔、工坊、仓库、研究台、农田）。消耗材料按血量缺口比例计算（向上取整）：

```
维修消耗 = ceil(建造材料 × (1 - 当前HP/最大HP))

示例1：围墙 Lv3（石砖×8 + 铁锭×2），当前 60% HP (360/600)
  → 石砖消耗 = ceil(8 × 0.4) = ceil(3.2) = 4
  → 铁锭消耗 = ceil(2 × 0.4) = ceil(0.8) = 1
  → 维修消耗：石砖×4 + 铁锭×1

示例2：防御塔 Lv3（石砖×8 + 铁锭×3 + 火药×3），当前 40% HP (200/500)
  → 石砖消耗 = ceil(8 × 0.6) = ceil(4.8) = 5
  → 铁锭消耗 = ceil(3 × 0.6) = ceil(1.8) = 2
  → 火药消耗 = ceil(3 × 0.6) = ceil(1.8) = 2
  → 维修消耗：石砖×5 + 铁锭×2 + 火药×2
```

维修动作为 1 回合。维修后建筑恢复满血。

### 3.7 英雄升级需求

| 等级 | 经验需求 |
|------|---------|
| 初级 → 熟练 | 100 经验 |
| 熟练 → 精英 | 250 经验 |

经验获取方式：采集/建造/PVE副本/完成悬赏任务。升级仅需经验，不需要额外材料。

### 3.8 采集与建造动作

#### 3.8.1 采集动作

采集/建造英雄可在资源地形格上执行"采集"动作。

```
动作流程：
  1. 英雄移动到资源地形格（相邻移动逐步进行）
  2. 发起采集指令 → 判题器校验 → 返回结果
  3. 持续采集 → 回合数走完 → 物品自动放入英雄背包
```

| 返回码 | 含义 | Agent 后续动作 |
|--------|------|---------------|
| `success` | 采集完成，物品已放入背包 | 继续采集或移动 |
| `in_progress` | 采集中，还需 N 回合 | 驻留继续 |
| `failed:inventory_full` | 背包满，物品无法放入 | 回库存放或丢弃物品 |
| `failed:resource_depleted` | 该格资源已枯竭 | 换位置采集 |
| `failed:not_gatherable` | 该格无可采集资源 | 换地形 |
| `failed:faction_locked` | 该资源对方独占（铜/硅） | 放弃，只能贸易获取 |

#### 3.8.2 建造动作

采集/建造英雄可在空地格上执行"建造"动作，持续 N 回合完成。

```
动作流程：
  1. 英雄移动到目标空地格
  2. 发起建造指令（含建筑类型） → 判题器校验 → 返回结果
  3. 每回合返回进度，完成后建筑落成
  4. 在进度完成后扣除背包材料
```

建造限制：
- **不可在已有建筑/围墙的格子上建造**
- 除此之外**无任何位置限制**——全图可建，不限制离基地距离，不限制中线

| 返回码 | 含义 | Agent 后续动作 |
|--------|------|---------------|
| `success` | 建造完成，建筑已落成 | 下一个动作 |
| `in_progress` | 建造中，还需 N 回合 | 驻留继续 |
| `failed:insufficient_materials` | 材料不足 | 先采集材料 |
| `failed:tile_occupied` | 目标格已有建筑/围墙 | 重新选位置 |

## 四、夜间防守系统

### 4.1 防御塔操控规则

```
白天：
  防御塔不工作（野兽不出没）

黑夜（80 回合）：
  兽潮从四面八方涌来
  最多 4 座防御塔，3 名英雄 → 英雄需不断换位操控不同方向的塔
  有英雄站在周围 1 格内 → 塔以面板伤害攻击射程内最近目标
  无英雄操控 → 塔不攻击

黎明：
  → 阳光彻底杀死地图上所有残余兽群
  → 阵亡英雄在基地复活
```

## 五、建筑系统

> 总共 **7 种建筑**：基地、研究台、工坊、仓库、农田、围墙、防御塔

### 5.1 建筑分类

| 分类 | 建筑 | 可否被摧毁 |
|------|------|:---:|
| 防御建筑 | 基地、围墙、防御塔 | **可被摧毁** |
| 功能建筑 | 研究台、工坊、仓库、农田 | **可被摧毁** |

> 所有建筑均可被摧毁，详见 [7.4 兽潮行为逻辑](#74-兽潮行为逻辑wc3塔防仇恨模型)
>
> **建筑重建规则**：HP 归零的建筑按**原等级**重建，消耗该等级对应的建造/升级材料（如 Lv3 围墙被毁 → 重建消耗石砖×8 + 铁锭×2），无需从 Lv1 重新逐级升级。**基地不可重建**——基地被毁直接淘汰。

### 5.2 基地（2×2）

| 属性 | 数值 |
|------|------|
| 等级 | Lv1（不可升级） |
| HP | 1500 |
| 防御力 | 15 |
| 功能 | ① 英雄复活点（阵亡次日清晨在此复活） |
|      | ② 提供夜间视野 + 友军视野共享 |
|      | ③ **淘汰判定核心**：基地被摧毁 → 玩家直接淘汰 |
| 建造 | 判题器在出生点预置，不可新建/移动 |

### 5.3 研究台（1×1）

| 等级 | 解锁内容 | 建造/升级材料 | HP | 防御力 |
|------|---------|---------|-----|-----|
| Lv1 | 步枪塔（防御塔 Lv3） | 木板×10 + 石砖×5 | 250 | 0 |

> 研究台仅 Lv1。激光塔（防御塔 Lv4）和合金城墙（围墙 Lv4）的图纸需通过悬赏任务获取。研究台可被兽群破坏，需按血量比例维修。

### 5.4 工坊（1×1）

| 等级 | 功能 | 建造/升级材料 | HP | 防御力 |
|------|------|---------|-----|-----|
| Lv1 | 基础加工：木板、石砖、面包 | 木材×8 | 350 | 3 |
| Lv2 | 加工速度+30%；解锁：铜锭、煤、铁锭、纯硅、火药、寿司、药膏 | 木板×10 | 400 | 5 |
| Lv3 | 加工速度+50%；解锁：纯铁、晶体管、电路板、芯片 | 石砖×8 | 500 | 8 |

### 7.4 兽潮行为逻辑

**核心原则：野兽的目标是基地，不是沿途单位。**

#### 规则一：路径优先（Path Priority）

野兽沿最短路径向基地移动。只要能前进就绝不攻击——哪怕路过防御塔旁边被射击。只有被**物理阻挡**（前方格子有建筑/围墙）时才停下来攻击阻挡物。

#### 规则二：阻挡触发仇恨（Block → Aggro）

当野兽无法前进时，攻击阻挡物以清出道路。仇恨**绑定在该阻挡物上**，直到阻挡物被摧毁，然后重新寻路攻击下一个阻挡物。

#### 规则三：仇恨转移条件

| 触发条件 | 行为 |
|---------|------|
| 当前阻挡物被摧毁 | 重新寻路，攻击路径上下一个阻挡物 |
| 英雄在野兽**邻格（1格）近战攻击** | 野兽转头还击英雄（"别惹我，惹我就干你"） |
| 英雄**离开邻格**（下一回合不在1格内） | 野兽不追击，回归规则一——继续向基地方向移动，攻击路径上的阻挡物 |
| 野兽1格内**同时有基地和英雄** | 野兽**直接攻击基地**——这是它的终极目标 |

#### 规则四：远程攻击不拉仇恨

防御塔（射程2格）和英雄远程攻击野兽时，野兽**不转移仇恨**。塔可以在墙后安全输出——前提是墙还立着。墙一破，塔暴露在野兽路径上，下一个被拆。

#### 规则五：围墙 Lv4 降攻速光环

围墙 Lv4（合金城墙）的降攻速光环（-50%，每 2 回合攻击 1 次）影响以其为中心 3×3 范围内的所有野兽。

#### 补充规则

- 防御塔需要英雄站在旁边（1 格内）操控才攻击
- **所有建筑均可被兽群破坏**，包括工坊/仓库/研究台/农田（不再是阻挡后才攻击——如果它们恰好挡在野兽最短路径上，野兽会拆掉它们）
- 黎明阳光杀死所有残余兽群
- **一方被淘汰后，兽潮数量减半**

### 7.5 防守失败后果

| 受损对象 | 后果 |
|---------|------|
| 围墙被破 | 野兽涌入攻击内部建筑 |
| 防御塔被毁 | 失去对应火力输出，需按原等级重建 |
| 功能建筑被毁 | 失去对应功能，需按原等级重建（仓库物品冻结，重建后恢复） |
| 基地被毁 | **玩家直接被淘汰**，本局游戏结束 |
| 英雄阵亡 | 次日在基地复活 |

## 九、悬赏任务系统

> 悬赏任务是让 LLM Agent 不可替代的核心设计。详见 `主要参考/agent_design.md`。

### 9.1 系统概述

世界消息中定期出现悬赏任务。任务通过**自然语言描述**，无精确坐标、无固定配方、无标准答案。开拓者使用**探索**和**使用**行动执行任务，Agent 负责：模糊寻路规划、自然语言线索解读、多英雄跨回合协调、失败后的动态重规划。

### 9.2 任务示例

| 简述 | 核心 Agent 能力 | 典型奖励 |
|------|:----------|---------|
| 城邦瘟疫，需制作特效药投湖 | 配方描述→物品映射、模糊寻路 | 积分+20、激光塔图纸×1 或 合金城墙图纸×1 |
| 石碑群，按正确顺序放置祭品 | 碑文→物品映射、顺序推理 | 积分+25、合金城墙图纸×1 或 合金城墙图纸×1 |
| 3卷残卷散落山脉，答题/满足诉求 | 谜题解答、诉求→物品映射、合成补全 | 积分+20、激光塔图纸×1 或 合金城墙图纸×1 |
| 流浪商人以物易物，描述古怪 | NL描述→物品映射(3个) | 积分+15、激光塔图纸×1 或 合金城墙图纸×1 |
| 异变矿脉需特殊防护才能采集 | 防护方法推理 | 积分+15、激光塔图纸×1 或 合金城墙图纸×1 |

### 9.3 任务共性结构

```
世界消息发布任务（NL描述，模糊地点）
  → 开拓者模糊寻路到达目标区域
  → 使用[探索]获取任务详情（NL线索，每局随机变化）
  → Agent推理：需要什么物品、以什么顺序、在什么位置
  → 多英雄配合：采集者收集材料、开拓者执行使用
  → 可能失败：祭品错误→被吞噬、谜题答错→冷却5回合
  → Agent决策：重试、放弃、还是换方案
```

### 9.4 图纸获取

每完成一个悬赏任务，奖励 **激光塔图纸×1** 或 **合金城墙图纸×1**（由任务内容决定）。一局比赛共 2 个任务，分别对应两种图纸，确保双方均有机会获取两种图纸。

### 9.5 任务刷新

- 每场比赛最多出现 **2 个**悬赏任务（从 5 种类型中随机选取）
- **第 1 天早上**刷新第一个任务，有效期至**第 6 天早上**
- **第 6 天早上**刷新第二个任务，有效期至**第 10 天结束**
- **双方玩家均可做同一任务，各自的任务地点不同，互不抢占**
- 任务难度随天数递增，后期任务奖励更丰厚

## 十、积分与排名系统

### 10.1 积分获取

| 行为 | 积分 | 说明 |
|------|------|------|
| 熬过 1 晚 | +10×天数 | 第 N 天天亮：+10N 分，如第1天+10、第2天+20...第10天+100 |
| 击杀小型兽 | +1 | 每只 |
| 击杀中型兽 | +2 | 每只 |
| 击杀大型/精英兽 | +4 | PVE 副本/兽潮 |
| 击杀 BOSS | +10 | 后期夜晚 |
| 清理废弃据点 | +5 | 每次清理成功 |
| **完成悬赏任务** | **+15~25** | 按任务难度 |

### 10.2 淘汰机制

- 如果**基地**被兽潮摧毁 → 玩家**直接淘汰**
- 淘汰后果：
  - 无法继续获取任何积分（已获得的积分保留）
  - 剩余天数不再参与，直接观战
  - 对方后续夜晚只承受自己一侧的兽潮

### 10.3 全局排名机制

- 每局 10 天结束后，双方各自累积的积分为**本局最终得分**
- 积分计入**全服跨对局大排行榜**（赛季制，如 1 个月/赛季）
- 积极合作贸易 → 双方都高分 → 大榜排名双高
- 积极完成悬赏任务 → 获取图纸+积分，拉开排名差距
- 排名奖励在赛季末结算

## 十二、完整游戏流程

### 12.1 开局初始状态（第 0 天）

| 类别 | 初始内容 |
|------|---------|
| 英雄 | 初级采集/建造英雄×2、初级开拓者×1 |
| 建筑 | **仅基地 Lv1（已建）**——其余建筑（工坊、仓库、围墙、防御塔、农田、研究台）均需玩家自行建造 |
| 物资 | 木材×60、石料×30、浆果×20 |
| 基地 | **已建，固定在出生点**，不可移动/新建。初始即拥有夜间视野。**基地是淘汰判定核心**——被摧毁则直接淘汰 |

> **设计意图**：开局只有基地，给玩家充分的布局自由度——围墙怎么围、塔放哪边、功能建筑摆哪，都是策略。初始物资足够建起基础防线（工坊+仓库+围墙+防御塔），但升级仍需采集。

### 12.2 每天流程

```
【晨间】
  → 系统推送当日新闻（含悬赏任务）
  → 玩家查看新闻、盘点库存、检查建筑状态
  → 昨夜阵亡英雄在基地复活

【白天 — 120 回合】
  → 采集英雄×2：采集资源 / 建造建筑 / 维修围墙 / 开垦农田 / 组队打副本
  → 开拓者：前往任务区域 / 使用[探索]获取线索 / 使用[使用]执行任务步骤 / 去贸易站交换

【黄昏】
  → 系统推送今夜兽潮预警
  → 检查围墙耐久、防御塔状态
  → 决定哪些英雄回防操控防御塔（每塔需要1人）

【黑夜 — 80 回合】
  → 兽潮来袭
  → 英雄站在塔旁操控攻击（不操控=塔不攻击）
  → 留在野外的英雄被野兽攻击 → 极大概率阵亡
  → 建筑受损 → 生成修复清单
  → 黎明 → 阳光杀死残余兽群

【次日】
  → 循环往复，至第 10 天结束
```

### 12.3 终局结算

- 第 10 晚兽潮结束后触发终局结算
- 统计双方积分（含悬赏任务积分）
- 积分上传全服大榜
- 生成本局数据报告

## 十四、设计亮点总结

1. **回合制驱动**：1天=200回合（白天120/黑夜80），10天=2000回合。每回合判题器↔玩家交互，Agent在关键节点（策略决策/任务规划/贸易谈判）介入
2. **阵营资源绑定 → 强制外交**：铜矿只有A能采、硅矿只有B能采，芯片必须通过贸易站交换获得
3. **渐进式合作**：Lv1→Lv2纯自给 → Lv2→Lv3自给（铁/硫磺双方均有）→ Lv3→Lv4必须合作（芯片=铜+硅）
4. **塔需英雄操控 → 自然回防**：最多4座塔、3名英雄，需不断换位操控不同方向的塔（射程统一2格），形成动态防守
5. **竞争性合作（无PVP）**：不能攻击对方，比的是"谁的运营更高效""谁的任务完成得更快"而非"谁打得更狠"
6. **图纸=悬赏任务唯一出口**：激光塔和合金城墙的图纸只能通过任务获取，不做任务永远卡在Lv3，Agent 参与不是可选项
7. **开拓者=Agent执行者**：悬赏任务不占用采集资源，是独立的Agent能力通道——模糊寻路、NL线索解读、多步执行、失败重规划
8. **英雄可复活、开局仅基地**：鼓励大胆博弈，不怕失误；开局仅基地，布局完全自由，所有建筑均需自行建造和保护
9. **建筑体系精简**：7 种建筑（基地2×2/其余1×1），25×25地图紧凑紧张
10. **日夜节奏清晰**：白天120回合探索/采集/任务，夜晚80回合集中回防，黎明阳光清场
