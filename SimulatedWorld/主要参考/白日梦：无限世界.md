# 《白日梦：无限世界》多 Agent 编程大赛 完整精简落地方案

## 一、赛事主题

白日梦・无限裂隙 —— 多智能体时空侦探协作对抗编程赛

依托 IP《白日梦：无限世界》无限流世界观，面向高校学生打造多 Agent 综合竞技赛事，融合大模型智能体、多智能体协同、动态 AI 生成副本、零固定剧本策略博弈，兼顾娱乐题材吸引力与计算机专业核心考核点。

## 二、赛事背景（极简版）

平行时空裂隙持续生成随机规则怪谈副本，每支队伍需自研 4 个差异化分工智能体化身时空侦探，在 AI 动态生成副本内完成线索解谜、资源采集、畸变 NPC 交互、裂隙封印；决赛阶段新增跨队伍 Agent 资源博弈、线索谈判竞争。

比赛环境由裁判引擎实时生成全新剧情、线索、随机事件，无固定关卡脚本，核心考验智能体局部观测通信、动态推理、自主任务分配能力。

## 三、参赛基础规则

1. 组队规模：1-4 人在校大学生（本科 / 高职 / 硕士均可）
2. 开发规范：主力 Python，支持 C++/Go 高性能辅助模块；官方统一提供环境调度 SDK、通信接口、环境观测 API
3. Agent 硬性配置：每队固定 4 个智能体，禁止增删、多开、复用角色
4. 两条技术赛道自由二选一
   - A 赛道 LLM 大模型 Agent：提示词工程、多智能体结构化通信、CoT 思维链、工具调用
   - B 赛道 MARL 多智能体强化学习：局部状态观测、联合决策、多智能体信用分配、动态环境适配

## 四、四类强制分工 Agent（固定职能闭环）

四智能体仅可通过官方受限通信信道传输结构化数据，带宽存在上限，无全局共享视野，强制训练信息压缩与关键信息筛选能力。

1. 侦查 Agent（探员）

   

   全域地图遍历、规则怪谈线索解析、危险标记识别、环境状态汇总，全队唯一全局观测采集单元。

2. 推理 Agent（谋士）

   

   聚合侦查原始数据、拆解多层级任务链、推演分支副本结局、生成全局最优行动策略、串联长线隐藏线索。

3. 行动 Agent（执行者）

   

   资源采集、机关解密、畸变怪物规避、裂隙封印核心操作，所有实体交互落地单元。

4. 交涉 Agent（调停者）

   

   NPC 多轮对话交互、敌方队伍智能体谈判博弈、道具线索交换、阵营选择、冲突调解。

## 五、比赛竞技场：AI 动态副本环境

### （一）副本生成机制

裁判内置叙事大模型，每轮对局实时生成独立副本，题材随机轮换：校园怪谈、古风古宅、科幻空间站、暴风雪山庄；内置随机突发事件、隐藏支线、可变 NPC 好感机制。

单局标准时长：8 分钟。

### （二）两大对局模式（分阶段启用）

1. 协作模式（初赛、复赛）

   

   单队 4Agent 独立闯关，核心目标封印主时空裂隙，解锁支线、隐藏结局获得额外积分；考核多智能体任务拆解、分工协同、资源调度。

2. 协作 + 跨队对抗模式（决赛）

   

   两支队伍共 8 个智能体同副本竞技，裂隙点位、道具资源总量有限；可合作交换线索，也可抢夺资源、误导敌方侦查、抢占封印优先权；考核多主体博弈、动态策略切换、风险决策。

## 六、完整赛事流程

1. 初赛（线上全自动评测）

   

   系统随机生成 20 套独立 AI 副本批量自动跑测，按总积分排名，前 60% 队伍晋级复赛。

2. 复赛（半线下天梯循环赛）

   

   混合协作、轻度对抗副本，采用 ELO 动态天梯积分排名，TOP16 队伍晋级线下决赛。

3. 决赛（现场可视化擂台）

   

   大屏实时展示副本场景、Agent 行动轨迹、跨智能体通信日志、队伍博弈过程；自动积分占 70%，现场技术答辩人工打分占 30%，综合决出奖项。

## 七、打分体系（自动 70% + 人工评审 30%）

### 1. 自动评测计分（全赛程生效）

加分项：主线裂隙封印、支线解谜 / 隐藏结局解锁、资源采集效率、完整生存时长；对抗模式额外奖励抢先封印、谈判道具收益。

扣分项：队内 Agent 行动冲突、通信冗余无效信息、重复无意义操作、触发副本灾厄导致全队重伤。

### 2. 人工评审计分（仅决赛）

智能体架构分工合理性、窄带宽通信方案设计、随机副本鲁棒性、创新算法 / 提示词框架、代码模块化工程规范、项目文档与可视化演示效果。

## 八、核心技术考察点

1. 多智能体系统：局部观测协同、分布式任务分配、轻量化自定义通信协议
2. LLM 智能体赛道：思维链推理、工具调用、多轮协商、大模型幻觉抑制
3. MARL 强化学习赛道：多智能体联合价值网络、探索 - 利用平衡、非平稳动态环境适配
4. 通用规划算法：时序任务拆解、多目标路径寻路、资源约束优化
5. 工程落地：智能体模块解耦、结构化状态传输、运行日志与可视化开发

## 九、赛事核心差异化亮点

1. IP 赋能：绑定高热度无限流 AI 游戏，规则怪谈题材贴合青年学生兴趣，降低参赛心理门槛
2. 双赛道兼容：零基础选手可快速搭建提示词 Agent 参赛，算法方向选手可深耕多智能体强化学习，兼顾入门与深度优化空间
3. 天然组队模式：4Agent 分工匹配 4 人宿舍组队，降低组队沟通成本
4. 无固定副本题库：AI 实时生成全新关卡，杜绝硬编码刷题，真实检验智能体通用推理能力
5. 强观赏性：决赛全流程可视化直播，直观展示 Agent 探索、谈判、对抗全过程
6. 技术覆盖面广：同时覆盖大模型应用、强化学习、分布式系统、规划算法多类计算机主流方向
