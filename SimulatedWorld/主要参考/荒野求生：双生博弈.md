# 荒野求生：双生博弈 完整游戏设定文档

# 1. 游戏背景

远洋中心坐落着与世隔绝的**蚀骨荒岛**，岛屿地形复杂、野兽横行、分布着废弃远古据点与稀缺矿产资源。荒岛开启常态化1v1生存竞技试炼，两名挑战者被空投至岛屿对角区域，从零开展荒野求生发育。

玩家不能暴力正面对抗，仅能通过资源抢占、市场垄断、信息欺诈、环境灾害、野兽引导、黑市特殊单位完成博弈压制。30个游戏自然日后，系统按照综合生存积分判定胜负；高分者赢得试炼全部奖励，低分者试炼失败，全队清空。

---

# 2. 全局基础对局规则

## 2.1 对局基础配置

- **对局模式**：纯1V1 Agent对抗，无第三方，双方出生点地图对角对称分布
- **对局周期**：总计30游戏天，1游戏天分为白昼、黄昏、夜间三个阶段；昼夜影响野兽行为、视野范围、作业效率
- **队伍人口上限**：全队所有NPC随从总数最高5名（含初始随从）
- **通用通货**：金币（交易、招募、黑市消费）、废料（高阶建筑、科技解锁核心硬通货）

## 2.2 人物基础四维属性

所有NPC随从统一拥有四项基础生存属性，属性过低会触发负面效果，属性归零随从永久死亡、不可复活：

1. **饥饿值**：工作行动持续消耗；通过浆果、兽肉、料理恢复；归零持续掉血

   **差异化数值上限**
   不同职业、不同等级的随从，饥饿值上限与基础消耗速率不同。体力消耗型职业上限更高、消耗更快；高阶随从能力更强，日常食物维护成本也显著高于初级随从，与成长体系形成平衡：

   - 等级基准上限：初级随从饥饿值上限 100 点；熟练随从上限 120 点；精英随从上限 150 点。
   - 职业消耗倍率（基础值）：采集工 / 猎手（重体力岗位）1.2 倍；工匠（中体力岗位）1.0 倍；斥候 / 医师（轻体力岗位）0.8 倍。
   - 等级消耗倍率：初级 1.0 倍；熟练 1.15 倍；精英 1.3 倍。

   **分级消耗规则**
   随从处于不同行为状态时，饥饿值消耗速度按强度分级：

   - 休息 / 睡眠状态：0.3 倍基础消耗，夜间在帐篷内休整时消耗最低
   - 轻体力工作（物资加工、制药、营地站岗）：1.0 倍基础消耗
   - 重体力工作（伐木采矿、建造施工、狩猎、长途外勤）：1.8 倍基础消耗
   - 战斗 / 应急奔袭状态：2.5 倍基础消耗

   **阈值联动效果**
   对齐饥荒经典饱食机制，饥饿值处于不同区间会触发对应的正面 / 负面效果，直接影响随从作战与工作效率：

   - 饱食状态（≥80%）：生命值缓慢自动回复，全工种工作效率提升 10%
   - 正常状态（30%~80%）：无额外加成与惩罚，可正常执行所有指令
   - 饥饿状态（<30%）：移动速度下降 20%，采集 / 建造效率下降 30%，停止自动回血
   - 濒死状态（数值归零）：每回合持续扣除生命值，直至补充食物或随从死亡

2. **生命值**：野兽攻击、腐坏食物、灾害损耗；通过药膏、医师技能恢复；归零永久死亡
3. **体力值**：采集、建造、外勤作业大量消耗；通过帐篷休息、高热量食物恢复；过低工作效率减半，强制停工

## 2.3 胜负结算规则

### 2.3.1 常规终局结算（30天到期）

综合生存积分 = 全队物资估值(35%) + 建筑总价值(30%) + 随从数量&等级质量(20%) + 探索/对抗任务加分(15%)，积分更高一方获胜。

### 2.3.2 提前结束对局

- 一方随从全部死亡，直接判负

# 3. 网格化地图与地形规则

## 3.1 地图规格

全图为**100×100标准1×1正方形网格**；单个地形格子为1×1通行单元；功能性建筑固定占用**4×4连续网格**；围墙、陷阱、火把、农田为1×1小型网格设施。

## 3.2 全图地形分类（固定属性）

| 地形类型 | 网格占比 | 通行&建造属性 | 核心资源产出 |
| --- | --- | --- | --- |
| 平坦草地 | 40% | 正常通行；可开垦农田；可搭建全部建筑（最优建家地形） | 浆果、野菜、农作物 |
| 密林区域 | 20% | 移速-30%；视野压缩50%；隐蔽性拉满；不可搭建大型建筑 | 木材、野生蘑菇、中小型野生动物 |
| 山地矿脉 | 15% | 移速-50%；视野+50%；仅可搭建简易防御设施 | 石料、金属矿、硫磺、金矿石 |
| 湖泊/河流 | 10% | 无法通行建造；沿岸格子可采集水资源 | 淡水、鱼类食材 |
| 固定废弃据点（8处） | 固定点位 | 可临时驻扎；不可长期建家；中后期核心争夺点位 | 废料、机械零件、随机稀有建筑图纸 |
| 固定高产矿场（6处） | 固定点位 | 可搭建采矿辅助建筑；多方博弈热点区域 | 高纯度金属、硫磺、金块 |
| 中心滩涂集市 | 1处全图唯一 | 全域中立安全区；禁止一切对抗、野兽不会入侵；黑市入口所在地 | 全品类物资交易、NPC招募、黑市特殊服务 |

## 3.3 领地规则

搭建并激活核心建筑【情报柜】后，周边20网格为己方专属领地；敌方无法在己方领地内搭建任意建筑。

# 4. NPC随从完整体系（核心角色系统）

## 4.1 基础选人&数量上限规则

- **开局选人**：对局初始化，玩家从5大职业中**自由选择2名初级随从**作为初始队伍
- **全局职业数量硬上限（全队生效）**：开局后可通过金币雇佣NPC，队伍NPC数量上限为采集工≤2、猎手≤2、工匠≤1、斥候≤1、医师≤1
- **成长体系**：全部随从三阶段成长：初级→熟练→精英；通过本职工作积累经验升级，也可消耗金币+材料快速晋升；等级越高日常食宿维护成本越高；随从死亡等级清零、永久消失

## 4.2 五大职业岗位职责&升级增益（全明细）

### 4.2.1 采集工（上限2名；资源&农耕核心）

**核心定位**：全链路原材料采集、农田开垦、物资搬运、基础防御施工

- **日常基础工作**：砍伐木材、开采石料；采集浆果/野菜/草药；湖泊搬运淡水；全队物资分类入库；库存余量上报
- **进阶工作**：开垦草地制作农田、播种浇水收割作物；矿场开采稀有矿石；废弃据点拆解杂物获取废料；修建维修围墙、陷阱
- **升级增益**：初级→熟练：采集速度+25%，负重+20%；熟练→精英：采集速度+30%，识别隐藏资源点；精英技能：高效开垦，农田总产量+50%

### 4.2.2 猎手（上限2名；食物&野外对抗核心）

**核心定位**：狩猎产粮、野外巡逻、队伍护航、野兽定向引导

- **日常基础工作**：野外狩猎获取兽肉、兽皮；营地周边布置狩猎陷阱；日常外围巡逻预警野兽；猎物初步加工运回仓库
- **进阶工作**：猎杀大中型野兽；清理周边野兽巢穴；护送外勤采集队伍作业
- **升级增益**：初级→熟练：狩猎伤害+30%，猎物掉落+20%；熟练→精英：狩猎伤害+40%；精英技能：引兽驱敌，定向引导野兽群移动至指定网格

### 4.2.3 工匠（上限1名；建造&生产核心）

**核心定位**：建材加工、全部建筑搭建、设施维修、工具装备生产

- **日常基础工作**：基础原材料深加工（木板、石砖）；制作工具、武器；搭建篝火、帐篷、储物箱；日常维修破损建筑设施
- **进阶工作**：搭建全部4×4功能核心建筑；升级建筑材质；批量制作高阶装备
- **升级增益**：初级→熟练：建造速度+30%，制作省15%材料；熟练→精英：建造速度+40%，制作省30%材料；精英技能：紧急抢修，1回合满血修复任意破损建筑

### 4.2.4 斥候（上限1名；信息&欺诈对抗核心）

**核心定位**：地图探索、敌方侦查、反侦察、虚假信息发布

- **日常基础工作**：探索未知地图，标记资源/矿点；侦查敌方营地布局；监控集市物价；营地周边反侦察排查
- **进阶工作**：潜入敌方周边探查库存、科技、人员编制；识别敌方伪装假营地
- **升级增益**：初级→熟练：视野+2网格，移速+20%；熟练→精英：视野+3网格，识破全部伪装；精英技能：散布流言，发布1条全局虚假暗影新闻

### 4.2.5 医师（上限1名；全队状态保障核心）

**核心定位**：药品制作、伤员救治、食材烹饪、全队健康风控

- **日常基础工作**：采集草药制作基础药膏；治疗受伤随从；烹饪食材制作高级料理；全天监控全队四维属性状态
- **进阶工作**：制作高阶疗伤、解毒药剂；营地环境防疫；处理灾害/野兽造成的重伤负面状态
- **升级增益**：初级→熟练：治疗效果+30%，草药采集+25%；熟练→精英：治疗效果+50%；精英技能：群体疫病免疫，全队免疫环境负面DEBUFF

# 5. 建筑系统

## 5.1 建筑规格划分

- 4×4大型核心功能建筑：全部生产、科技、生存主建筑，受领地规则保护，禁止敌方手动破坏
- 1×1小型辅助建筑：围墙、陷阱、火把、农田，可被野兽破坏、可被敌方间接破坏

## 5.2 建筑分类及功能

### 5.2.1 生存保障类

篝火、帐篷、烹饪锅、储水器；负责照明、属性恢复、食物加工、淡水储备

### 5.2.2 生产加工类

工作台、熔炉、晾肉架、鞣革架；原材料深加工、工具装备生产

### 5.2.3 科技领地类（核心建筑）

研究台、情报柜；情报柜激活领地保护；研究台解锁高阶图纸与科技

### 5.2.4 防御预警类

材质围墙、尖刺陷阱、瞭望塔；抵御野兽入侵、提前预警敌方动向

### 5.2.5 情报对抗类

信号塔、伪装棚；获取隐秘新闻、隐藏己方营地布局

# 6. 环境、野兽与光照机制

## 6.1 天气与环境系统

游戏内随机触发晴天、暴雨、酷暑、台风四类天气；会影响采集效率、建筑耐久、物资腐坏速度；全部天气异动由每日官方新闻提前预告。

## 6.2 野兽全局硬性规则

1. **通用趋光规则（最高优先级）**：全地图所有等级野兽，**不会进入任何光照覆盖网格；有光区域100%驱离全部野兽，无例外**
2. **攻击目标限制**：野兽仅攻击**无光照区域的NPC随从、无光照区域建筑**；不会偷窃物资、不会破坏仓库库存物品
3. **野兽分级**：小型（野鼠、暗影狐）、中型（野猪、暗影狼）、大型（礁岛巨蜥）；仅伤害数值、建筑破坏能力有区别，趋光规则完全一致
4. **行为逻辑**：白天仅在巢穴周边游荡，不主动入侵；夜间全域游荡主动搜寻无光目标发起攻击

# 7. 黑市特殊对抗单位（小偷+杀手，无违规惩戒）

两者为滩涂黑市单次雇佣中立单位，不属于己方随从，执行任务后消失；无任何违规处罚、无红名惩戒，仅亏损雇佣成本。

## 7.1 黑市小偷

- 雇佣限制：单局最多雇佣3次；夜间黑市开放招募
- 执行规则：**敌方任意NPC视野范围内无法执行偷窃**；仅能偷窃敌方1×1露天储物箱普通资源；无法触碰情报柜核心物资与图纸；单次偷窃上限对方露天库存10%
- 失败后果：潜行失败直接撤离，仅扣除佣金

## 7.2 暗影杀手

- 雇佣限制：单局最多雇佣2次；黑市限量刷新，高额金币+废料招募
- 执行触发条件：**不分白天黑夜、不受光照限制；目标周边2网格范围内敌方随从总数≤1即可执行刺杀**
- 执行效果：刺杀成功目标NPC永久死亡；无溯源痕迹；刺杀失败杀手直接逃逸
- 约束：无任何违规惩戒、无积分扣除；仅承担雇佣成本亏损风险

# 8. 新闻事件驱动系统

每日清晨全图同步推送中立「荒岛纪事」新闻，是天气、物价、怪物、资源异动的唯一触发源；新闻不标注数值变动，需要Agent自主推理研判影响。分为三个可信度等级：

1. **官方公告（100%真实）**：灾害预警、集市税务、兽潮公告；确定全局数值变动
2. **滩涂传闻（70%真实）**：矿脉异动、遗迹刷新；大概率生效，细节存在偏差
3. **暗影流言（50%真实）**：可由己方精英斥候主动发布虚假消息；用于信息欺诈误导对手

# 9. 四大类核心1V1对抗机制

全部对抗不触碰顶层铁律，无直接伤人、无手动拆核心建筑

## 9.1 资源&经济对抗

- 抢占高价值矿场、废弃据点等核心网格资源点
- 结合新闻物价预判，囤货居奇垄断刚需物资，抬高对手采购成本
- 集市批量扫货卡位核心原材料，卡住敌方科技、建筑升级节奏
- 低价倾销打压敌方主营变现物资，打崩对手资金链

## 9.2 环境&野兽对抗

- 破坏敌方外围光照设施，放任野兽入侵敌方无光区域
- 精英猎手定向引导野兽、全图兽潮冲击敌方外围营地
- 灾害天气破坏敌方防灾设施，放大天灾对敌方营地的损耗

## 9.3 信息&伪装对抗

- 斥候侦查敌方全部发育、人员、布局情报
- 搭建假营地、假物资堆伪装自身实力
- 发布虚假暗影流言，诱导对手错误搬迁、盲目囤货

## 9.4 灰色单位对抗

- 派遣黑市小偷窃取敌方露天储备资源
- 满足条件雇佣暗影杀手，清除敌方核心高阶功能性随从

# 10. Agent 决策中枢：定位+标准化全流程+实例化任务

## 10.1 Agent核心定位

Agent为队伍唯一决策大脑；**不执行任何实操动作**；仅完成信息接收、决策研判、下发指令、复盘迭代；统筹全队生存、发育、对抗全部行为。

## 10.2 Agent固定六步标准化决策链路（通用全场景）

1. **感知输入**：汇总随从状态、库存、建筑、新闻、敌方动向、告警信息
2. **意图识别**：识别核心诉求，按「生存风险＞日常发育＞博弈对抗」判定任务优先级
3. **目标拆解**：将大目标拆分为可落地、带时间节点的子任务
4. **行为规划**：人员分工、路线规划、资源测算、风险兜底预案
5. **指令下发**：对对应职业NPC下发明确可执行工作指令
6. **结果迭代**：核验任务完成度，复盘偏差，优化后续决策方案

## 10.3 Agent每日固定例行工作

- 晨间：全局态势盘点、解读当日新闻、判定物价走势
- 上午：给两名及以上随从下发全天分工任务
- 日间：实时监控突发风险、敌方动向，动态调整指令
- 黄昏：营地安全巡检，检查夜间光源、防御、物资储备
- 夜间：当日全流程复盘，制定次日发育对抗策略

## 10.4 全场景实例化Agent落地任务（可直接执行）

### 10.4.1 L1基础生存任务：对局第1天从零搭建合规安全营地

**场景**：开局2名初始随从（采集工+工匠）；白昼8小时倒计时天黑；周边草地+密林+湖泊；目标天黑完成光照闭环，夜间野兽无法入侵。

**Agent完整执行链路**：

1. 感知输入：初始物资、双随从满状态、地形分布、昼夜倒计时、无新闻无风险
2. 意图识别：最高优先级生存任务；核心目标：夜间光源防野兽 > 基础工具制作 > 温饱保障
3. 目标拆解：制作基础工具→采集建材食材→搭建篝火+帐篷→留存整夜燃料
4. 行为规划：采集工外勤就近采集资源；工匠原地制作工具+搭建建筑；测算材料阈值；兜底：优先保篝火光源，延后帐篷建造
5. 指令下发：明确两名随从采集顺序、建造节点、返程时间
6. 结果迭代：入夜核验营地安全度，优化次日采集通勤路线

### 10.4.2 L1应急任务：野外重伤随从救援

**场景**：猎手单独狩猎重伤，残血滞留密林，临近黄昏野兽风险飙升

**Agent链路**：识别人员死亡最高优先级→拆解救人+运回物资→规划安全救援路线→调度医师+采集工外勤接应→下发集结指令→复盘优化组队狩猎规则

### 10.4.3 L2进阶对抗任务：抢占中央硫磺矿+垄断市场套利

**场景**：新闻预告7天后火药物资涨价40%；双方同时争夺中心高产硫磺矿

**Agent链路**：感知双方距离、物价走势→识别资源垄断博弈高优目标→拆解抢占采矿+集市扫货+控价出货→规划护卫采矿分工、对抗兜底预案→下发侦查/采矿/警戒指令→涨价周期复盘盈利与压制效果

### 10.4.4 L2信息对抗任务：发布假流言诱导敌方营地搬迁

**场景**：敌方资源匮乏具备搬迁意向；己方精英斥候可发布虚假流言

**Agent链路**：感知敌方发育态势→识别信息欺诈低成本对抗目标→拆解现场伪装+假新闻发布+市场造势→规划全套伪装流程→下发多随从协同指令→监控敌方行为验证欺骗效果

### 10.4.5 L2环境对抗任务：引导兽潮冲击敌方营地

**场景**：官方新闻预告跨区域兽潮；敌方位于兽潮原生路径

**Agent链路**：感知兽潮参数、敌方营地弱点→识别天灾转嫁对抗目标→拆解破坏敌方光源+布置引导诱饵+己方营地防御→下发潜行+引兽+防守指令→战后侦查敌方损失复盘战术效果

### 10.4.6 L2极端决策任务：雇佣杀手清除敌方落单高阶随从

**场景**：终局阶段己方积分落后；敌方精英采集工长期单人外勤作业，满足刺杀条件

**Agent链路**：感知积分差距、目标行动规律、雇佣成本→风险收益意图研判→拆解侦查锁定目标+雇佣杀手+抢占后续矿点→制定执行与兜底方案→下发黑市雇佣+后续抢占指令→复盘对局积分反超效果
