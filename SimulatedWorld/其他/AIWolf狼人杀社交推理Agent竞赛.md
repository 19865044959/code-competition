# AIWolf 狼人杀 / 社交推理 Agent 竞赛

> **类型**：学术竞赛 / 不完全信息博弈 / 社交推理
> **来源**：日本人工智能学会 (JSAI) + 国际学术社区
> **最新赛事**：第6届 AIWolf 国际竞赛 (2024)；ICML 2025 LSPO 论文
> **关键词**：狼人杀、社交推理、欺骗检测、不完全信息、自然语言协商

---

## 一、核心理念

将 AI Agent 投入经典的**狼人杀 (Werewolf/Mafia)** 游戏中。Agent 必须通过自然语言讨论、推理、欺骗来获胜——这是AI最困难的挑战之一：需要同时维持**真实与虚假两种内部状态**，并理解其他Agent如何感知自己。

---

## 二、狼人杀游戏规则（5人标准版）

| 角色 | 人数 | 目标 | 特殊能力 |
|------|------|------|---------|
| 村民 (Villager) | 2 | 找出并投票放逐狼人 | 无 |
| 预言家 (Seer) | 1 | 帮助村民找出狼人 | 每晚查验一人身份 |
| 狼人 (Werewolf) | 1 | 隐藏身份，存活到最后 | 每晚可以杀一人 |
| 狂人 (Possessed) | 1 | 帮助狼人获胜（但不知道狼人是谁）| 无 |

**游戏流程**：
```
夜晚 → 白天讨论（辩论+投票） → 放逐 → 夜晚 → ... 
直到狼人被放逐（村民胜）或狼人数量≥村民（狼人胜）
```

---

## 三、核心 AI 技术挑战

### 3.1 "骗子困境" (Liar's Duality)
- 狼人 Agent 必须**同时维护**：
  - 内部真实状态：我知道我是狼人
  - 公开伪装状态：我是一个普通的村民
- 还要**推理其他 Agent 的推理**：
  - "他认为我是什么角色？"
  - "他认为我认为他是什么角色？"（二阶信念）

### 3.2 信息不对称
- 预言家知道真相但不能直接暴露（会被狼人杀）
- 狼人知道同伴但必须假装不知道
- 狂人想帮狼人但不知道谁是狼人

### 3.3 最新技术方案（2024-2025）

| 方法 | 论文/赛事 | 核心思路 |
|------|----------|---------|
| **RL-instructed LLM** | NeurIPS 2024 | RL训练"讨论策略选择器"→LLM生成自然语言 |
| **LSPO (CFR+DPO)** | ICML 2025 | 在策略空间做博弈论优化→对齐到LLM语言空间 |
| **多Agent信念建模** | AIWolf 2024 | 每个Agent维护"谁是谁"的概率分布 |

## 四、作为 Agent 比赛的改编设计

### 赛制建议：《暗流都市》

```
世界观：一座赛博朋克城市，潜伏着"异种"(Werewolf)
角色扩展：
- 市民 ×N（村民）
- 侦探（预言家，拥有监控权限）
- 异种 ×M（狼人）
- 叛徒（狂人，帮助异种但不知道它们是谁）
- 黑客（可修改一次查验结果）
- 医生（可保护一人免于暗杀）

比赛形式：
- 每局 8-12 名 Agent（由多支队伍提供）
- 连续打 N 局，统计各队伍 Agent 胜率
- 加分项：逻辑推理质量、说服力、欺骗成功率

评分维度：
- 胜率（按角色加权）
- 推理准确性（是否正确识别了异种）
- 欺骗质量（作为异种时的存活天数）
- 语言质量（自然度、策略性、一致性）
```

### 核心考验点

| 维度 | 考察内容 |
|------|---------|
| **信念建模** | 对他人身份的概率推断 |
| **策略性语言** | 什么时候说什么、怎么说 |
| **欺骗与反欺骗** | 制造谎言vs识别谎言 |
| **联盟政治** | 拉拢摇摆票、孤立对手 |
| **风险管理** | 作为预言家何时跳身份 |

### 差异化优势

- **极高观赏性**：狼人杀本身就是热门直播内容
- **成熟生态**：AIWolf 社区、数据集、评估框架可直接使用
- **角色可扩展**：可自定义新角色、新机制
- **中文天然优势**：现有比赛以日文/英文为主，中文狼人杀存在巨大空白
