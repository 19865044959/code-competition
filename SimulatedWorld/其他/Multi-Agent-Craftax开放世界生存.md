# Multi-Agent Craftax — 开放世界生存协作

> **类型**：学术基准 / 开放世界 MARL
> **来源**：Oxford + University of Waterloo (arXiv:2511.04904)
> **前身**：Crafter (单Agent 开放世界)
> **加速**：JAX GPU 原生加速，2.5亿步/小时
> **关键词**：开放世界、程序生成、生存协作、交易系统、异质Agent

---

## 一、核心理念

受《我的世界》(Minecraft) 启发，但完全用 JAX 重写以实现**GPU级超高速并行**。支持**任意数量 Agent** 在程序生成的开放世界中**采集、建造、打怪、贸易**。

---

## 二、两个变体

### Craftax-MA：开放式多Agent生存
| 特性 | 说明 |
|------|------|
| Agent数 | 任意（1-N） |
| 世界 | 程序生成（每次不同） |
| 核心机制 | 资源采集、工具制作、建造、战斗 |
| 目标 | 开放式（生存+发展） |
| 速度 | 2.5亿环境步/小时（单GPU） |

### Craftax-Coop：异质Agent协作生存
| 特性 | 说明 |
|------|------|
| **异质Agent** | 矿工、采集者、战士——三个不同角色 |
| **交易系统** | Agent之间可以交换资源 |
| **生命/复活机制** | Agent会受伤，需要队友救助 |
| **必需协作** | 单人无法完成所有任务——强制合作 |

---

## 三、与《荒野求生：双生博弈》的对照

| 机制 | 荒野求生 | Craftax |
|------|---------|---------|
| 地图 | 100×100 固定 | 程序生成 |
| Agent数 | 1+5 NPC随从 | N个独立Agent |
| 职业系统 | 5职业（采集/猎手/工匠/斥候/医师） | 3角色（矿工/采集者/战士） |
| 交易 | 中心集市 | Agent间直接交易 |
| 对抗 | 1v1 | 协作或对抗均可 |
| 加速 | 无 | GPU JAX加速 |

### 启示
- Craftax 的**交易系统**可以添加到荒野求生的 1v1 中
- Craftax 的**异质Agent设计**验证了"强制协作"的可行性
- **GPU 加速**是让大规模评测可行的关键技术

## 四、作为 Agent 比赛的改编设计

### 赛制建议：《异星殖民地》— 协作生存挑战

```
世界观：一颗资源丰富但严酷的外星行星

每队：3-5 个异质 Agent（矿工/采集者/战士/医生/工程师）
环境：程序生成的 128×128 地图
周期：100 游戏天

目标：建立可持续的殖民地，在以下灾难中幸存：
- 资源枯竭（某区域矿脉采尽）
- 外星野兽袭击（类似荒野求生的野兽机制）
- 极端天气（沙尘暴/辐射风暴）
- 疾病爆发（需要医生Agent研发解药）

评分：
- 殖民地总人口（生存为先）
- 科技等级（解锁了多少技术）
- 资源储备（食物/水/能源库存）
- 基础设施（建筑数量与等级）

可选的对抗模式：
- 两张地图两个殖民地 → 先达到"发射救援信号"的获胜
- 共享一张大地图 → 资源竞争但不直接攻击
```

### 核心考验点

| 维度 | 考察内容 |
|------|---------|
| **分工协作** | 异质Agent之间的任务分配 |
| **资源优化** | 稀缺资源的最优分配 |
| **危机响应** | 突发灾难的应急处理 |
| **长期规划** | 100天的可持续发展策略 |
| **适应新环境** | 程序生成地图的泛化能力 |

### 差异化优势
- **硬件加速**：GPU模拟让大规模穷举评测可行
- **程序生成**：杜绝"背图"问题
- **强制协作**：单人无法生存，设计师可调"协作必须度"
- **科幻题材**：区别于"荒野求生"的现实题材
