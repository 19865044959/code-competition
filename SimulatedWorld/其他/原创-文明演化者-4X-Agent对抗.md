# 原创比赛设计：《文明演化者》— 4X 文明 Agent 对抗大赛

> **灵感来源**：Project Doxa + Aivilization + Sid Meier's Civilization
> **类型**：原创比赛设计
> **关键词**：文明演化、4X策略、长周期、涌现叙事、多维度胜利

---

## 一、世界观

一片名为**"起源大陆"**的神秘土地上，多个原始部落同时觉醒。每个部落由一个**AI 文明领袖**统治，下面是若干个**AI 公民**。文明的命运由这些 AI 的集体智慧决定——他们能否从蛮荒走向星辰大海？

---

## 二、游戏规模

| 参数 | 数值 |
|------|------|
| 文明数 | 4-8 个 |
| 每文明人口 | 10-50 Agent |
| 地图 | 128×128 六边形网格 |
| 回合数 | 500 回合（覆盖从石器时代到信息时代） |
| 单局时长 | 2-4 小时 |
| 自然语言 | Agent 间外交/贸易使用自然语言 |

---

## 三、4X 机制

### Explore（探索）
- 程序生成地图：山地、河流、森林、沙漠、海洋
- 蛮族营地：中立敌对单位
- 自然奇观：特殊地块提供独特加成
- 其他文明：外交接触的起点

### Expand（扩张）
- 建立城市（需选择位置、规划布局）
- 人口增长（食物供给驱动）
- 边界扩张（文化影响力驱动）
- 殖民新大陆

### Exploit（开发）
- 资源采集系统（类似 Aivilization 的物品系统）
- 科技树（从农业到AI共 6 个时代）
- 文化政策（军事/经济/科技/文化四个方向）
- 奇观建造（一次性全球独有的大型项目）

### Exterminate（征服）
- 军事单位（战士→骑兵→坦克→机甲）
- 围城战（城市防御力 + 地形加成）
- 联盟战争（多文明联合行动）
- 核威慑（信息时代解锁）

---

## 四、AI 文明领袖的决策空间

每个文明由一个**领袖 Agent** 统领，需要做出以下类型的决策：

```
每回合决策循环：
1. 态势感知：地图、资源、外交、军力、科技、文化
2. 目标设定：短期（10回合）/ 中期（50回合）/ 长期（200回合）
3. 策略选择：战争？和平发展？科技冲刺？文化输出？
4. 资源分配：人口分配到哪些岗位？资金投向哪里？
5. 外交行动：宣战/和平/同盟/贸易协定/间谍
6. 公民指令：给下属 AI 公民下达具体任务
```

---

## 五、胜利条件

| 胜利类型 | 条件 | 需要的文明特质 |
|----------|------|--------------|
| **征服胜利** | 消灭所有其他文明的首都 | 军事优势 + 战略时机 |
| **科技胜利** | 率先完成"星际殖民"计划 | 科技领先 + 资源充裕 |
| **文化胜利** | 文化影响力覆盖所有文明 | 文化产出 + 开放外交 |
| **经济胜利** | 控制全球 60% 的 GDP | 贸易垄断 + 资源控制 |
| **外交胜利** | 获得全球议会 2/3 投票 | 外交技巧 + 信誉积累 |
| **时间胜利** | 500 回合结束时综合评分最高 | 全面发展 |

## 六、比赛赛制

```
初赛：Round Robin
  - 4 文明局 × N 轮
  - 每队在不同地图、不同出生点各打一轮
  - 积分：胜利 +3，平局 +1，失败 +0

复赛：Swiss System
  - 同积分的队伍对战
  - 8 文明大地图
  - 前 4 名晋级

决赛：All-Star 大乱斗
  - 8 个文明同图竞技
  - 大屏直播 + 实时文明演化可视化
  - 邀请真人嘉宾解说（类似体育赛事）
```

## 七、核心技术考察点

| 维度 | 考察内容 |
|------|---------|
| **长周期策略** | 500 回合的超长规划能力 |
| **多目标优化** | 军事 vs 经济 vs 科技 vs 文化的权衡 |
| **外交博弈** | 何时结盟、何时背叛、复杂多极格局 |
| **战争策略** | 兵力部署、地形利用、后勤补给 |
| **信息不完全** | 敌方真实实力被"战争迷雾"隐藏 |
| **自适应对手** | 面对不同风格的文明领袖采取不同策略 |

---

## 八、差异化优势

1. **深度与广度**：500 回合 × 4 维度胜利条件 → 策略空间巨大
2. **涌现叙事**：每局产生独特的"世界历史"
3. **观赏性顶级**：文明版图变化的可视化类似"历史地图演变"
4. **教育价值**：可以融入真实历史元素
5. **无限可重玩**：程序生成地图 + 多类型AI对手 = 无唯一解
6. **社区价值**：赛后可以生成"本局世界史"纪录，成为社区内容

---

## 九、技术可行性

- 回合制 → 不需要实时物理模拟
- 地图规模可控（128×128）
- Agent 数量可控（每文明 10-50）
- 可使用 JAX 加速模拟（类似 Craftax）
- 视觉展示可以用 2D 地图（不必须 UE5）
