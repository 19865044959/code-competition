# 原创比赛设计：《异星殖民地》— 多Agent协作生存大赛

> **灵感来源**：Multi-Agent Craftax + 荒野求生：双生博弈 + RimWorld/Factorio
> **类型**：原创比赛设计
> **关键词**：协作生存、程序生成、异质Agent、资源管理、灾难应对

---

## 一、世界观

**地球联邦**启动了"播种计划"——向系外行星派遣殖民飞船。每艘飞船载有若干 AI 管理的殖民者，他们必须在严酷的外星环境中白手起家，建立可持续的殖民地。

然而，**多个殖民地降落在同一颗行星上**。行星资源有限，殖民地的存亡取决于：是选择合作还是竞争？

---

## 二、核心设计

### 2.1 殖民地配置

每个参赛队伍管理一个殖民地，包含：

| 角色 | 数量 | 职责 |
|------|------|------|
| **殖民地总督 (Agent)** | 1 | 全局决策、资源分配、外交 |
| **工程师** | 2-3 | 建造、维修、科技研发 |
| **矿工** | 2-3 | 资源采集、勘探 |
| **农民** | 2-3 | 食物生产、畜牧 |
| **医生** | 1-2 | 伤病治疗、疾病防控 |
| **警卫** | 1-2 | 防御外星生物、维持秩序 |

共计 **9-14 个单位**，其中总督是 LLM Agent，其余可以是规则/RL驱动的 NPC。

### 2.2 五维资源

| 资源 | 获取方式 | 用途 |
|------|---------|------|
| **食物** | 农业、狩猎、贸易 | 防止饥饿 |
| **水** | 河流/湖泊采集、净化 | 生存必需、农业 |
| **能源** | 太阳能、地热、核聚变 | 建筑运转、科技研发 |
| **矿物** | 采矿、回收 | 建造、工具、武器 |
| **知识** | 研究、探索遗迹 | 解锁新科技、新建筑 |

## 三、灾难系统

行星不友好。以下灾难随机触发（由新闻系统提前预警）：

| 灾难 | 影响 | 应对方式 |
|------|------|---------|
| **沙尘暴** | 太阳能 -80%，室外作业 -50% 效率 | 提前储能、转为地热 |
| **外星兽潮** | 攻击殖民者、破坏建筑 | 围墙+防御塔+警卫 |
| **瘟疫** | 殖民者持续减血 | 医生研发疫苗 |
| **地震** | 随机建筑损毁 | 工程师抢修 |
| **陨石** | 随机地块被毁 | 建立防护罩科技 |
| **心理危机** | 殖民者效率 -30%，可能叛变 | 建造娱乐设施、心理辅导 |

## 四、协作 vs 对抗

### 4.1 初赛：纯协作模式

```
目标：建立可持续殖民地，达到"自给自足"标准
评分：
- 总人口（生存率）
- 科技等级
- 资源储备（食物/水/能源的缓冲天数）
- 基础设施评分
- 灾难存活率
```

### 4.2 决赛：竞争+协作混合

```
地图：共享大地图（256×256）
殖民地：4-8 个（分布在地图不同区域）
中央区域：全球独有的高产资源、外星遗迹

冲突点：
- 中央高产区归谁？（先到先得 vs 协商分配）
- 有限的外星遗迹（科技爆发点）
- 贸易（互通有无 vs 技术封锁）

可选的零和博弈：
- 决赛可以选择"竞争模式"：最先发射返航飞船的殖民地获胜
- 或者"共存模式"：全部殖民地都存活的情况下，评分最高者获胜
```

## 五、Agent 决策架构

每个**殖民地总督 Agent** 运行以下循环：

```
回合开始
  ↓
感知阶段：读取状态（资源、人员、建筑、灾难预警、邻居动态）
  ↓
优先级排序：生存风险 > 资源保障 > 科技发展 > 外交竞争
  ↓
策略制定：本回合的核心目标
  ↓
任务拆解：将目标拆分为各职业NPC的具体任务
  ↓
指令下发：对每个NPC类型下发任务指令
  ↓
执行反馈：观察执行结果，修正下一轮决策
```

## 六、技术实现

### 6.1 环境模拟

| 参数 | 方案 |
|------|------|
| 地图 | 256×256 网格（Python 原生） |
| 物理 | 简化的格子世界（不需要UE5） |
| Agent API | Python SDK + REST API |
| 加速 | 多进程并行模拟 |
| 可视化 | 2D 地图 + 实时数据面板 |

### 6.2 单局参数

| 参数 | 数值 |
|------|------|
| 游戏天数 | 100 天（≈ 现实 90 分钟） |
| 每天步数 | 黎明/白天/黄昏/夜晚 4 阶段 |
| Agent 决策频率 | 每阶段 1 次（每天 4 次） |

## 七、核心考验点

| 维度 | 考察内容 |
|------|---------|
| **资源优化** | 有限资源在多目标之间的最优分配 |
| **危机管理** | 突发灾难的应急响应 |
| **长周期规划** | 100天的科技树路径、设施布局 |
| **劳动分工** | 异质NPC的高效调度 |
| **协作/竞争平衡** | 什么时候合作（贸易）、什么时候竞争（抢占） |
| **风险应对** | 安全边际 vs 激进发展 |

---

## 八、差异化优势

1. **生存压力驱动策略**：灾难系统确保没有"最优解"，必须动态适应
2. **协作到竞争的平滑过渡**：初赛培养合作能力，决赛考验博弈智慧
3. **RimWorld式的叙事魅力**：每个殖民地都有独特的故事
4. **可调节难度**：灾难频率、资源丰度都是可调参数
5. **中文科幻题材**：区别于西方主流的"中世纪奇幻"或"僵尸末日"
