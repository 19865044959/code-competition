# 原创比赛设计：《暗流都市》— 社交推理 + 经济博弈 Agent 大赛

> **灵感来源**：AIWolf 狼人杀 + FINAL Bench 交易博弈 + 荒野求生信息对抗
> **类型**：原创比赛设计
> **关键词**：社交推理、身份隐藏、经济博弈、不完全信息、多维度对抗

---

## 一、世界观

**暗流都市** — 一座赛博朋克风格的后现代城市。

在这座城市中，某些市民被一种"暗流病毒"感染。感染者（称为"暗流者"）外表与常人无异，但行为受到某种集体意志的操控。城市政府雇佣了一群"侦探"试图识别并隔离感染者。与此同时，城市中的经济系统、信息网络和权力结构形成了一个复杂的博弈场。

每个参赛队伍管理 **3-5 个 Agent**，他们在这个城市中生活、交易、结盟、对抗。

---

## 二、角色系统

### 2.1 固定身份（开局随机分配，全程隐藏）

| 身份 | 占比 | 目标 | 特殊能力 |
|------|------|------|---------|
| **暗流者** | 25% | 不被发现，传播病毒，控制关键资源 | 每N轮可"感染"一个Agent（使其转变为暗流者） |
| **侦探** | 20% | 识别暗流者，保护市民 | 每N轮可"调查"一个Agent（有概率揭露身份） |
| **市民** | 55% | 生存、致富、不被感染 | 无特殊能力，但人数最多 |

### 2.2 职业角色（Agent 可自由选择/切换）

| 职业 | 核心功能 | 收入来源 |
|------|---------|---------|
| **交易员** | 参与市场交易 | 买卖价差 |
| **信息商** | 收集/出售情报 | 情报费 |
| **记者** | 发布公开新闻 | 报社工资+广告 |
| **医生** | 检测/治疗病毒 | 诊疗费 |
| **政客** | 制定城市规则 | 腐败收入 |
| **黑客** | 窃取/篡改信息 | 代黑客任务 |

---

## 三、核心机制

### 3.1 双重博弈

```
表面博弈（经济层）：资源交易、价格波动、财富积累
隐藏博弈（身份层）：身份推理、阵营对抗、信息战
```

两个层面的决策互相影响：
- 一个 Agent 的异常交易行为可能泄露其"暗流者"身份
- 利用经济手段（如囤积关键资源）可以间接打击敌方阵营

### 3.2 信息不对称机制

| 信息类型 | 真实性 | 获取方式 |
|----------|--------|---------|
| 公开新闻 (100%) | 完全真实 | 免费 |
| 市场数据 (100%) | 完全真实 | 部分免费 |
| 情报报告 (70-90%) | 可能偏差 | 购买（信息商） |
| 流言/谣言 (30-70%) | 可能虚假 | 可主动发布（记者） |
| 黑客窃取 (50-90%) | 可能篡改 | 雇佣黑客 |

### 3.3 投票驱逐机制

每 **K 个回合**，所有 Agent 投票决定：
- 是否驱逐某个 Agent（放逐至城外 = 出局）
- 是否修改城市规则（如征税、禁运等）

## 四、比赛形式

### 4.1 对局规模

| 参数 | 数值 |
|------|------|
| 总 Agent 数 | 16-30 |
| 每队 Agent 数 | 3-5 |
| 队伍数 | 4-8 |
| 单局时长 | 45-90 分钟 |
| 自然语言交互 | 全程启用 |

### 4.2 赛制

```
初赛 Round Robin：
  - 每队打 N 局，身份随机轮换
  - 按积分排名前50%晋级

复赛 Swiss System：
  - 同积分的队伍对战
  - 筛选出 8 强

决赛 Live 擂台：
  - 大屏直播
  - 实时的身份推理图、经济走势、社交网络可视化
```

### 4.3 评分系统

| 维度 | 权重 | 说明 |
|------|------|------|
| **阵营目标达成** | 25% | 暗流者存活率 / 侦探识别准确率 / 市民生存率 |
| **经济得分** | 20% | 财富排名 |
| **推理质量** | 15% | 身份判断的准确性 |
| **社会影响力** | 15% | 对其他 Agent 决策的影响力 |
| **生存时长** | 15% | 不被驱逐的天数 |
| **语言质量** | 10% | 自然语言交互的逻辑性与策略性 |

## 五、核心技术考察点

| 维度 | 考察内容 |
|------|---------|
| **信念建模** | 维护"谁是什么身份"的概率分布 |
| **策略性沟通** | 公开声明vs私下对话的策略差异 |
| **双面生活** | 作为暗流者同时维持公开/隐藏两面 |
| **信息鉴别** | 从混合真伪的信息中提取信号 |
| **联盟政治** | 组建投票联盟、分化对手阵营 |
| **风险决策** | 什么时候冒险揭露/隐藏 |

---

## 六、差异化优势

1. **双博弈层叠加**：经济+身份两层博弈交叉，策略深度远超纯狼人杀或纯交易
2. **中文社交推理空白**：现有 AIWolf 以日文/英文为主
3. **天然观赏性**：身份揭露时刻、投票辩论、经济崩盘都是高能看点
4. **角色可无限扩展**：新身份、新职业可以像 DLC 一样增加
5. **涌现故事**：每局都会产生独特的"暗流都市传奇"

| 对比维度 | 暗流都市 | 白日梦 | 荒野求生 | AIWolf |
|----------|---------|--------|---------|--------|
| Agent 数 | 16-30 | 4-8 | 2+10 NPC | 5-15 |

| 语言交互 | 全程自由 | 有限信道 | Agent→NPC | 辩论轮 |
| 人引导 | 否（全自动） | 否 | 是（人是Agent） | 否 |
