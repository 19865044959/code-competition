# Project Doxa — 文明演化 Agent 沙盒

> **类型**：开源项目 / 文明模拟 / LLM Agent
> **来源**：GitHub - SpaceCypher/doxa
> **关键词**：文明演化、涌现行为、信任与战争、信仰系统、OODA循环

---

## 一、核心理念

AI Agent 作为**文明公民**生活在演化世界中。Agent 形成文明、建造建筑（神庙、兵营、谷仓）、担任角色（祭司、士兵、农民、长老），资源稀缺驱动贸易、合作与冲突。

**核心特色**：拥有一个"上帝模式"层，实验者可以注入饥荒、瘟疫、神迹或大规模思想操纵，观察社会如何反应。

---

## 二、核心机制

### 2.1 Agent 认知架构

```
OODA 循环：
  Observe（观察环境、资源、其他Agent）
  → Orient（解释信息、匹配信仰）
  → Decide（选择行动：采集/建造/贸易/攻击）
  → Act（执行并记录结果）
```

### 2.2 文明动力学

| 机制 | 说明 |
|------|------|
| **资源** | 木材、水、石头、黄金动态生成；稀缺性驱动行动 |
| **建筑** | 神庙（信仰中心）、兵营（军队）、谷仓（食物存储）|
| **角色** | 祭司（传播信仰）、士兵（作战）、农民（采集）、长老（决策）|
| **信任图** | Agent之间/文明之间的信任网络 |
| **战争触发** | 文明间信任低于阈值 → 自动宣战 |

### 2.3 信仰系统
- Agent 可以**发明和传播信仰**
- 祭司传播信仰效果更强
- 集体信仰影响整个文明的文化走向

### 2.4 阿萨比亚指数 (Asabiyyah Index)
- 衡量社会凝聚力（灵感来自 Ibn Khaldun 的历史理论）
- 低值 → 内部冲突、可能触发内战
- 受资源压力、外部威胁、领袖能力影响

### 2.5 "上帝模式" (Demiurgic Layer)
操作者可注入：
- **自然灾害**：饥荒、瘟疫、洪水
- **神迹**：丰收、治愈
- **思想注入**：大规模传播仇外心理/和平主义
- **干预目的**：观察社会对冲击的反应

## 三、涌现行为观察

根据官方文档和社区反馈，已观察到的涌现现象：

1. **自组织分工**：Agent 根据环境条件和群体需求自发分化角色
2. **贸易网络形成**：不同文明根据资源禀赋建立贸易路线
3. **信仰分裂与融合**：文明内可能出现宗教分裂，或跨文明信仰融合
4. **战争-和平周期**：类似历史的兴衰循环
5. **文化认同形成**：长期共存的 Agent 产生独特的"文化风格"

## 四、作为 Agent 比赛的改编设计

### 赛制建议：《文明演化者》— 4X 文明 Agent 对抗

```
世界观：一片资源丰富但有限的大陆
参赛者：
- 每队部署 1 个"文明领袖 Agent" + N 个"公民 Agent"
- 总共 4-8 个文明同图竞技

周期：500 回合（≈ 2000 年的文明演化）

目标：
- 文明存续（未被消灭）
- 文化影响力（信仰传播范围）
- 科技水平（建筑数量与等级）
- 经济实力（资源总量与贸易规模）

对抗机制：
- 领土争夺（使用地图系统）
- 资源竞争（市场定价博弈）
- 信仰竞争（宗教传播对抗）
- 军事征服（宣战与联盟体系）
- 外交博弈（条约、贸易协定、同盟）

胜利条件：
- 统治胜利：消灭所有对手
- 文化胜利：信仰覆盖 80% 人口
- 科技胜利：率先完成"奇观"建造
- 经济胜利：控制 60% 全球GDP
```

## 五、核心考验点

| 维度 | 考察内容 |
|------|---------|
| **长周期策略** | 500回合尺度上的规划与适应 |
| **多维度博弈** | 军事+经济+文化+外交多维竞争 |
| **联盟动态** | 何时结盟、何时背叛、多极格局 |
| **资源优化** | 有限资源下的增长与分配 |
| **危机管理** | 自然灾害/战争/内乱中的响应策略 |

### 差异化优势
- **文明尺度**：比个体 Agent 博弈更高一个层次
- **历史叙事**：每局都会产生独特的历史故事
- **教育价值**：相当于一个"历史模拟器"
- **技术深度**：涉及 RL+LLM+博弈论多方面技术
