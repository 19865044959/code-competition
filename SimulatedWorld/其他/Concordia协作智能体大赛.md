# Concordia 协作智能体大赛 (NeurIPS 2024)

> **类型**：学术竞赛 / LLM 多Agent协作对抗
> **来源**：Google DeepMind + Cooperative AI Foundation
> **赛事**：NeurIPS 2024 官方竞赛
> **框架**：Concordia Library v2.0 (开源)
> **关键词**：LLM Agent、社会困境、协作博弈、自然语言协商

---

## 一、比赛核心理念

将 LLM Agent 置于**社会困境 (Social Dilemma)** 场景中，Agent 之间通过**自然语言自由交互**，展现协作能力：承诺守信、谈判协商、互惠互利、声誉管理、伙伴选择、妥协让步、惩罚制裁。

**核心洞察**：现实世界中，合作不靠全局控制器，而靠 Agent 之间的语言沟通。这个比赛直接测试 LLM 的"协作智能"。

---

## 二、比赛框架 (Concordia Library)

### 2.1 技术架构

```
Game Master (GM) → 描述场景、裁决行动
Agent 1 ←→ Agent 2 ←→ Agent 3 (自然语言对话)
      ↕
   Environment (资源池、公共品、惩罚机制)
```

- **GM** 是一个 LLM，负责生成场景描述、解析 Agent 行动、裁决结果
- **Agent** 是参赛者开发的 LLM Agent，通过自然语言与其他 Agent 和 GM 交互
- **Environment** 模拟公共品博弈、资源困境等经典社会困境

### 2.2 典型场景

| 场景类型 | 描述 | 协作挑战 |
|----------|------|---------|
| **公共品博弈** | N 个 Agent 决定投入多少资源到公共池 | 搭便车问题 |
| **资源困境** | 共享有限资源（如渔场） | 公地悲剧 |
| **信任博弈** | 先付出信任，对方选择回报或背叛 | 信任建立 |
| **囚徒困境** | 双方同时选择合作或背叛 | 策略推理 |

## 三、比赛形式

### 赛制

```
初赛：单场景协作评分
  - 多个社会困境场景
  - 评估：协作成功率、集体收益、个体收益
  
复赛：混合动机对抗
  - 同一场景中既有共同利益又有竞争
  - 评估：能否在竞争中保持合作底线

决赛：多Agent自由交互
  - 复杂社会网络环境
  - 评估：涌现的协作结构、联盟形成
```

### 评分机制

| 指标 | 权重 | 说明 |
|------|------|------|
| 集体总收益 | 30% | 所有 Agent 收益之和 |
| 个体收益 | 20% | 己方 Agent 自身收益 |
| 信任维持率 | 20% | 承诺兑现比例 |
| 声誉评分 | 15% | 其他 Agent 对己方的信任度 |
| 鲁棒性 | 15% | 在不同对手/场景下的一致性 |

## 四、核心技术考察点

1. **协商策略**：如何出价、让步、达成协议
2. **承诺机制**：如何让对手相信你会兑现
3. **声誉管理**：如何积累和维护信任
4. **合作伙伴选择**：识别可靠/不可靠的对手
5. **惩罚与制裁**：如何威慑背叛行为
6. **自然语言说服**：用语言（而非数学策略）影响他人

---

## 五、对"白日梦"赛制的启示

| Concordia | 可借鉴点 |
|-----------|---------|
| GM 裁判机制 | 你的"裁判引擎"可以用 LLM 做类似的场景生成和裁决 |
| 自然语言 Agent 交互 | 白日梦中交涉Agent的设计可以直接参考 |
| 社会困境设计 | 可设计"线索共享vs独自封印"等两难选择 |
| 协作+竞争共存 | 初赛协作、决赛对抗的模式 |

---

## 六、技术可行性

- **Concordia 库完全开源**，可直接基于它搭建原型
- **LLM API 调用**是主要成本（按 token 计费）
- **单局时长**：约 5-15 分钟（取决于对话轮数）
- **可扩展性**：支持 2-20+ Agent 同时交互
