## 二、核心发现

### 2.1 关键趋势

1. **从单 Agent 到多 Agent 协作+对抗**：NeurIPS 2024-2025 的竞赛趋势明显向多 Agent 系统转移
2. **LLM Agent 成为主流**：传统 RL 与 LLM Agent 融合，自然语言交互成为新范式
3. **开放式生成环境**：不再使用固定脚本/关卡，AI 动态生成副本和环境
4. **人机协作维度**：不仅 Agent 之间对抗，人引导 Agent 的"协同智能"成为新方向
5. **区块链/Token 经济融合**：Web3 + AI Agent 竞赛模式兴起（FoxLeague, AI GRANPRIX）

### 2.2 市场空白与机会

| 空白领域 | 描述 | 差异化潜力 |
|----------|------|-----------|
| **中文社区 Agent 比赛** | 现有比赛几乎全是英文/日文 | ⭐⭐⭐⭐⭐ |
| **混合人机协作模式** | 现有比赛要么纯 Agent 要么纯人，人引导 Agent 中间地带少 | ⭐⭐⭐⭐⭐ |
| **长周期生存博弈** | 多数比赛是短时单局制，缺少跨天/跨周的持久战 | ⭐⭐⭐⭐ |
| **开放式世界+多维度对抗** | 多数比赛对抗维度单一（纯经济或纯战斗），缺少复合维度 | ⭐⭐⭐⭐ |
| **可观赏性设计** | 多数研究基准无观赏性考虑，缺少直播友好设计 | ⭐⭐⭐⭐ |

### 2.3 与你现有 3 个 Case 的对照

| 你的 Case | 最接近的竞品 | 差异化空间 |
|-----------|-------------|-----------|
| **Aivilization（模拟社会）** | Stanford Smallville, Project Doxa | 更丰富的经济系统、多人引导 |
| **白日梦：无限世界（4Agent协作对抗）** | Concordia Contest, Lux AI | 动态AI生成副本、IP叙事包装 |
| **荒野求生：双生博弈（1v1生存）** | Craftax-MA, Overcooked | 长周期、多维度对抗（经济+信息+环境） |
