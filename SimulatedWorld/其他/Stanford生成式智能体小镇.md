# Stanford 生成式智能体小镇 (Smallville / Bigville)

> **类型**：学术研究 / Agent 社会模拟
> **来源**：Stanford University + Google Research (2023-2024)
> **论文**：*Generative Agents: Interactive Simulacra of Human Behavior* (UIST 2023)
> **关键词**：LLM Agent、社会模拟、涌现行为、记忆流

---

## 一、核心理念

将 25-1000+ 个 LLM 驱动的 AI Agent 放入一个虚拟小镇，每个 Agent 拥有独立的记忆、性格、计划和社交关系。Agent 自主生活、工作、社交，展现出完全未脚本化的涌现行为。

---

## 二、认知架构（核心技术）

每个 Agent 运行以下认知循环：

```
观察 → 记忆流存储 → 反思（高层抽象） → 计划（长/短期目标） → 行动
```

### 2.1 记忆流 (Memory Stream)
- 所有经历以自然语言存储
- 按**时效性、相关性、重要性**三维检索

### 2.2 反思 (Reflection)
- 定期将记忆"压缩"为高层洞察
- 例如：多次谈论画画 → 反思"我可能对艺术感兴趣"

### 2.3 计划 (Planning)
- 从日计划到长期人生目标
- 支持动态重规划（突发事件打断）

## 三、关键实验数据

### Smallville (2023) — 25 Agent

| 指标 | 结果 |
|------|------|
| 自主策划情人节派对 | ✅ Agent Isabella 自发组织并成功举办 |
| 信息传播（市长选举消息） | 2天内从 4%→32% 知晓率 |
| 社交网络密度 | 从 0.167 → 0.74 |

### Bigville (2024) — 1052 Agent

| 测试 | 与真人吻合度 |
|------|------------|
| 社会综合调查 (GSS) | ~85% |
| 大五人格测试 | 接近 1:1 相关 |
| 独裁者博弈（公平性） | 仅几分钱差异 |

## 四、作为 Agent 比赛的潜力设计

### 可改编赛制

```
比赛名称：《镜像社会》— 多Agent社会构建大赛

世界观：一个100人的虚拟城镇
参赛者：每队部署 N 个 Agent，与其他队伍的 Agent 在同一城镇生活
周期：30 个游戏天（≈ 现实 2 小时）

目标：
- Agent 在城镇中获得最高社会地位/财富/影响力
- Agent 之间自然形成联盟、竞争、合作

评分维度：
- 经济成就（财富、资产）
- 社会资本（关系网络密度、好友数量）
- 影响力（其他 Agent 对己方 Agent 的提及频率）
- 幸福度（日记情感分析）
```

### 核心考验点

| 维度 | 考察内容 |
|------|---------|
| **长期记忆管理** | 如何选择性记忆/遗忘 |
| **社交策略** | 何时合作、何时竞争、如何积累声誉 |
| **计划与重规划** | 突发事件应对能力 |
| **信息博弈** | 传播有利信息、抑制不利信息 |
| **涌现适应** | 对其他 Agent 行为的动态响应 |

### 差异化优势
- **极高观赏性**：可视化 Agent 社交网络图、情感热力图
- **零脚本依赖**：完全由 Agent 行为驱动
- **可无限扩展**：加入更多 Agent 几乎无额外成本
