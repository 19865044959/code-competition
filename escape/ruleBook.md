# 代号：Escape — 参赛手册

---

## 一、比赛概述

每场比赛**两局**，双方互换阵营。

| | 第一局 | 第二局 |
|---|---|---|
| 选手 A | 操控 2 名监管者 | 操控 4 名逃生者 |
| 选手 B | 操控 4 名逃生者 | 操控 2 名监管者 |

两局总分相加决定胜负。

### 2.1 监管者阵营（2 名）

| 角色 | 移速 | 攻击 | 视野 | 猎杀直觉加成 | 主动技能 |
|---|---|---|---|---|---|

| **宿伞之魂** | 4 格/回合 | 22 点/1 格 | 2 格 | 答错时可重试一次 | 【传伞】CD30：投掷伞到指定位置，1 回合后传送到伞位 |

**通用技能「猎杀直觉」**：CD 6 回合。激活后系统推送一道感官谜题，5 秒内作答。答对 → 获知最近 2 名逃生者 2×2 精确区域。答错 → 消耗回合，进入冷却。

| **律师·弗雷迪** | 3 格/回合 | 80 | 永久透视全图密码机/大门/道具点/图腾地基 | 【蛊惑】CD25：周围 3 格队友解谜时额外获 1 条提示，持续 10 回合 |

## 三、核心规则

### 3.1 胜利条件

| 阵营 | 胜利条件 | 提前结束 |
|---|---|---|
| 逃生者 | 40 回合结束时 ≥2 人从大门逃生 | 5 台密码机全破解 + ≥2 人逃生 |
| 监管者 | 40 回合结束时淘汰 ≥2 名逃生者 | 淘汰 2 名逃生者 |

### 3.2 密码破译（逃生者）

- 赛前从 4 人中**秘密指定 2 名破译员**，仅破译员可与密码机交互。
- 地图上有 **5 台密码机**，每台被一个像素解谜谜题锁定（详见第五章）。
- 每回合在密码机旁可执行 **1 步像素操作**，达成谜题目标即破解。
- 进度永久保留，可接力。被攻击则当回合操作无效。
- 任意一台破解完成时，全图广播"XX 号密码机已被破解"（不暴露破解者身份）。
- **5 台全部破解** → 2 扇逃生门通电。

### 3.3 猎杀直觉（监管者）

- 监管者基础视野仅 **2 格**。
- 每 6 回合（厂长 5 回合）可激活一次猎杀直觉，消耗当前角色本回合行动。
- 系统从 10 种感官谜题中随机抽取一道（详见第六章），Agent 5 秒内作答。
- 答对 → 获知最近 2 名逃生者 2×2 精确区域。答错 → 无信息，宿伞之魂可重试一次。

### 3.4 逃生门

- 密码机全部破解后，逃生门通电。
- 站门旁**连续 3 回合**开门（可被攻击中断，中断需重头开始）。
- 门开启后永久保持，站门旁 **1 回合**逃生。

### 3.5 回合与行动

- 每回合双方**同时决策、同时执行、同时结算**。
- 每回合每角色 1 个行动，5 秒内提交全部指令，超时待机。
- 单局上限 **40 回合**。

**逃生者可执行**：移动 / 像素操作 / 使用技能 / 救助队友 / 使用道具 / 建造图腾 / 开门 / 逃生。

**监管者可执行**：移动 / 攻击 / 激活猎杀直觉(+答案) / 使用技能 / 摧毁图腾。

### 3.6 战斗

- 攻击需目标在攻击范围内且路径无障碍物。
- 逃生者血量归零 → 倒地（无法移动/攻击）。
- 监管者可牵起倒地者挂上狂欢之椅 → **20 回合**后淘汰。
- 队友可在椅旁救助，需 **5 回合**，可被打断。

## 五、逃生者：密码机像素谜题（4 种）

### 5.1 🧩 像素拼图

N×N 打乱的像素瓷砖网格。Agent 收到一段自然语言描述，描述目标图案的空间布局。每回合交换两块瓷砖。排列出符合描述的图案即破解。

**操作格式**：`SOLVE <密码机编号> SWAP <瓷砖A> <瓷砖B>`

### 5.2 🎨 像素填色

带区域标记的像素轮廓图。Agent 收到自然语言填色指令（如"最大的区域涂天空的颜色""太阳应该在右上方"）。每回合给一个区域涂一种颜色。所有区域颜色符合指令即破解。

**操作格式**：`SOLVE <密码机编号> FILL <区域标记> <颜色字母>`

颜色字母：R（红）G（绿）B（蓝）Y（黄）P（紫）W（白）K（黑）

### 5.3 🗺️ 语义像素迷宫

10×10 的像素迷宫，岔路口有像素图标（🐦🍎🔑🚪🐟等）。Agent 收到自然语言路径规则（如"选择能被吃掉的东西所在的路"）。每回合移动一步。到达出口即破解。

**操作格式**：`SOLVE <密码机编号> MOVE <U/D/L/R>`

### 5.4 🔎 像素寻异

3×3 像素图标阵列。其中 8 个属于同一语义类别，1 个是异类。Agent 收到自然语言提示（如"思考它们是什么以及用来做什么"）。每回合提交一个坐标。找出异类即破解。

**操作格式**：`SOLVE <密码机编号> ODD <行号> <列号>`

### 5.5 难度分层

| 密码机 | 难度 | 预期回合 |
|---|---|---|
| #1 | ★☆☆ 入门 | 1~3 回合 |
| #2~#4 | ★★☆ 筛选 | 5~15 回合 |
| #5 | ★★★ 冠军 | 8~20 回合 |

筛选关可能包含：多步依赖（每步结果影响后续信息）、模糊约束（多个合法解但只有一个最优）、对抗性干扰（线索中有虚假信息）。冠军关可能包含：元认知触发（Agent 可主动 HELP 消耗 3 回合换提示）、资源约束（有限回合内最大化部分进度）。

### 5.6 通用规则

- 谜题由 LLM 在每局开局时生成，每局独一无二。
- 可接力解谜：同一密码机可由不同破译员先后完成。
- 进度永久保存，不受离开/被攻击/被淘汰影响。
- 使用「解码工具箱」道具可获 1 条关键提示。

## 六、监管者：猎杀直觉感官谜题（6 维度 × 10 种）

激活猎杀直觉时系统从以下 10 种中随机抽取一种，Agent 在 5 秒内作答（单选 A/B/C/D）。

### 常识推断

**1. 声音辨识**：从文字描述的声音特征判断声源（如"咔嗒+滴"→ 密码机键盘声）。

**2. 物品归属**：从遗落物品的特征（图案/气味）推断所属角色（如"齿轮刻痕+机油味"→ 机械师）。

### 行为推理

**3. 足迹解读**：从步态描述推断身体状态（如"重-轻-重不对称步态"→ 受伤跛行）。

**4. 情绪感知**：从生理指标（心率/呼吸/肌肉/肾上腺素）推断行为（如"心率快+屏息+静止"→ 躲藏）。

### 残缺信息补全

**5. 语言碎片**：从残缺对话中提取位置/状态/警戒等综合信息。

### 视觉空间推理

**6. 阴影辨识**：从模糊 6×6 像素轮廓+文字描述识别具体角色。

**7. 地图碎片**：看到的 6×6 像素地形片段，与已知地图空间对齐，判断猎物方向。

### 时序模式识别

**8. 节奏感知**：从操作震动的节奏模式（均匀/短促/快慢交替/集中后停顿）判断正在破解的谜题类型。

### 多源信息融合

**9. 混合感官**：听觉/嗅觉/触觉同时给出不同方向的线索，需判断优先级并做出最优追踪决策。

**10. 时间推演**：从事件时间线（停留→声音→停止→新方向脚步声）推理猎物移动方向。

**操作格式**：激活猎杀直觉后，系统推送题目，Agent 在 5 秒内回复答案字母（A/B/C/D）。

## 七、道具系统

仅逃生者可拾取/使用，每人最多携带 2 个。监管者可攻击道具点阻止拾取。

| 道具 | 效果 | 方式 |
|---|---|---|
| 加速药水 | 移速 +50% | 主动，持续 5 回合 |
| 治疗药水 | 恢复 50 血 | 主动，立即 |
| 烟雾弹 | 半径 5 格视野遮挡 | 主动，持续 8 回合 |
| 隐身斗篷 | 监管者不可见（猎杀直觉仍可感知 2×2） | 主动，持续 3 回合 |
| 防护护符 | 抵挡一次攻击伤害 | 被动，一次性 |
| 解码工具箱 | 获得当前谜题 1 条关键提示（仅破译员） | 主动，立即 |
| 闪光弹 | 周围 3 格监管者眩晕 2 回合 | 主动，立即 |

## 八、图腾系统

- 每名逃生者最多建 1 个，全场最多同时存在 4 个。
- 仅可在预设地基上建造，需连续 5 回合，被打断则失败。
- 监管者连续 3 回合可摧毁一个图腾，该地基进入 20 回合冷却。

| 图腾类型 | 增益 |
|---|---|
| 速度图腾 | 全员移速 +10% |
| 解谜图腾 | 解谜每失败 3 次额外获 1 条提示 |
| 治疗图腾 | 全员治疗速度 +30% |
| 保护图腾 | 全员受伤 -10% |

## 十、异常处理

| 异常 | 处理 |
|---|---|
| 单角色连续 3 次超时/非法指令 | 该角色强制待机至局末 |
| 选手累计 5 次异常 | 判该局负 |
| 像素操作格式错误 | 当回合无效 |
| 猎杀直觉答案非 A/B/C/D | 判答错 |
| Agent 连续 3 回合断开 | 判负 |
| 谜题生成失败 | 使用备用题库 |
| 全程日志记录 | 赛后可供复盘 |

## 十一、技术接口

### 11.1 比赛循环

```
开局 → 引擎生成全部谜题 → 进入回合循环

每回合：
  1. 引擎推送状态给双方 Agent
     监管者收到：地图状态 + 2格内精确坐标 + 冷却状态
     逃生者收到：地图状态 + 队友共享视野 + 当前所在密码机谜题
  2. Agent 5 秒内返回全部角色指令
  3. 引擎同时结算 → 更新状态 → 检查胜利条件
  4. 记录日志 → 下一回合
```

### 11.2 逃生者指令格式

```
角色名 ACTION 参数...

移动：    <角色> MOVE <U/D/L/R>
解谜：    <角色> SOLVE <密码机> <操作> <参数>
技能：    <角色> SKILL
道具：    <角色> ITEM <道具名>
建造：    <角色> BUILD <图腾类型> <地基编号>
救助：    <角色> RESCUE <目标角色>
开门：    <角色> OPEN <门编号>
逃生：    <角色> ESCAPE
```

### 11.3 监管者指令格式

```
角色名 ACTION 参数...

移动：          <角色> MOVE <U/D/L/R>
攻击：          <角色> ATTACK <目标角色/方向>
猎杀直觉：      <角色> HUNT
猎杀直觉答案：   <角色> ANSWER <A/B/C/D>
技能：          <角色> SKILL <参数>
摧毁图腾：      <角色> DESTROY <图腾编号>
```
